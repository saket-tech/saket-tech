- Set up Docker-based deployment and CI/CD with GitHub Actions for AWS EC2 releases.
- Integrated MetaMask authentication, IPFS storage, and secure hosting with Caddy SSL.

### Full-Stack Blog Application
Responsive blog platform using React, Node.js, Express.js, and PostgreSQL.
